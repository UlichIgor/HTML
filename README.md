# HTML5

## Теги HTML, упорядковані в алфітному порядку

### Основні теги
```html
<!-- * `<code></code>` *Визначає коментар*. -->
<!-- * `<code>!DOCTYPE</code>` *Визначає тип документа*. -->
<!-- * `<code><html></html></html></code>` *Визначає корінь документа **HTML***. -->
* `<code>&lt;head&gt;&lt;/head&gt;</code>` *Містить метадані/інформацію для документа*.
* `<code>&lt;body&gt;&lt;/body&gt;</code>` *Визначає тіло документа*.
```
### Текстові теги
```html
* `<code>&lt;a&gt;&lt;/a&gt;</code>` *Визначає гіперпосилання*.
* `<code>&lt;abbr&gt;&lt;/abbr&gt;</code>` *Визначає абревіатуру або акронім*.
* `<code>&lt;address&gt;&lt;/address&gt;</code>` *Визначає контактну інформацію для автора/власника документа*.
* `<code>&lt;b&gt;&lt;/b&gt;</code>` *Визначає жирний текст*.
* `<code>&lt;bdi&gt;&lt;/bdi&gt;</code>` *Відокремлює частину тексту, яка може бути відформатована в іншому напрямку, від іншого тексту поза ним*.
* `<code>&lt;bdo&gt;&lt;/bdo&gt;</code>` *Перевизначає поточний напрямок тексту*.
* `<code>&lt;blockquote&gt;&lt;/blockquote&gt;</code>` *Визначає розділ, цитований з іншого джерела*.
* `<code>&lt;cite&gt;&lt;/cite&gt;</code>` *Визначає назву твору*.
* `<code>&lt;code&gt;&lt;/code&gt;</code>` *Визначає частину комп'ютерного коду*.
* `<code>&lt;del&gt;&lt;/del&gt;</code>` *Визначає текст, який було видалено з документа*.
* `<code>&lt;dfn&gt;&lt;/dfn&gt;</code>` *Визначає термін, який буде визначено у вмісті*.
* `<code>&lt;em&gt;&lt;/em&gt;</code>` *Визначає виділений текст*.
* `<code>&lt;i&gt;&lt;/i&gt;</code>` *Визначає частину тексту іншим голосом або настроєм (курсив)*.
* `<code>&lt;ins&gt;&lt;/ins&gt;</code>` *Визначає текст, який було вставлено в документ*.
* `<code>&lt;kbd&gt;&lt;/kbd&gt;</code>` *Визначає введення з клавіатури*.
* `<code>&lt;mark&gt;&lt;/mark&gt;</code>` *Визначає позначений/виділений текст*.
* `<code>&lt;pre&gt;&lt;/pre&gt;</code>` *Визначає попередньо відформатований текст*.
* `<code>&lt;q&gt;&lt;/q&gt;</code>` *Визначає коротку цитату*.
* `<code>&lt;s&gt;&lt;/s&gt;</code>` *Визначає текст, який більше не є правильним*.
* `<code>&lt;samp&gt;&lt;/samp&gt;</code>` *Визначає зразок виведення з комп’ютерної програми*.
* `<code>&lt;small&gt;&lt;/small&gt;</code>` *Визначає менший текст*.
* `<code>&lt;span&gt;&lt;/span&gt;</code>` *Визначає розділ у документі*.
* `<code>&lt;strong&gt;&lt;/strong&gt;</code>` *Визначає важливий текст*.
* `<code>&lt;sub&gt;&lt;/sub&gt;</code>` *Визначає підрядковий текст*.
* `<code>&lt;sup&gt;&lt;/sup&gt;</code>` *Визначає надрядковий текст*.
* `<code>&lt;u&gt;&lt;/u&gt;</code>` *Визначає підкреслений текст*.
* `<code>&lt;var&gt;&lt;/var&gt;</code>` *Визначає змінну*.
```

```html
* `<track>` *Визначає текстові доріжки для медіа-елементів (**<video></video> і **<audio></audio>**)*.
```

### Непідтримувані теги
```html
* `<acronym></acronym>` Не підтримується в **HTML5**. *Замість цього використовуйте `<abbr></abbr>` для визначення акроніма*.
* `<big></big>` Не підтримується в **HTML5**. *Натомість використовуйте **CSS** для визначення великого тексту*.
* `<center></center>` Не підтримується в **HTML5**. *Натомість використовуйте **CSS** для визначення центрованого тексту*.
* `<font></font>` Не підтримується в **HTML5**. *Натомість використовуйте **CSS** для визначення шрифту, кольору і розміру тексту*.
```

## Довідник атрибутів HTML

### Загальні атрибути

*В ***HTML*** є кілька атрибутів, які є загальними для всіх елементів, такі атрибути звуться **загальними**, **ґлобальними** чи **універсальними***.

*Ці атрибути можуть використовуватися з усіма тегами, хоча на деякі з них вони можуть ніяк не впливати*.

*Всі зазначені нижче атрибути є загальними і можуть бути вказані до будь-якого **HTML** елемента*.
```html
* `accesskey="key"` *Дозволяє активувати або сфокусуватись на елементі за допомогою заданого сполучення клавіш*.
* `class="classname"` *Визначає один або кілька класів, щоб зв’язати елемент з таблицею стилів (**CSS**)*.
* `contenteditable="true|false"` *Дозволяє редагувати текст в середині тегу*.
* `contextmenu="menu_id"` *Контекстне меню для елемента*.
* `dir="ltr|rtl"` *Вказує напрямок тексту*.
* `draggable="true|false"` *Визначає чи можна перетягувати елемент*.
* `dropzone="copy|move|link"` *Вказує, що робити з перетягуваними елементами*.
* `hidden` *Приховує вміст елемента від перегляду*.
* `id="unique_id"` *Задає унікальний ідентифікатор для елемента*.
* `lang="language_code"` *Визначає основну мову вмісту елемента*.
* `spellcheck="true|false"` *Вказує браузеру перевіряти чи ні правопис і граматику в тексті елемента*.
* `style="CSS_styles"` *Задає вбудований (**inline**) ***CSS*** стиль для елемента*.
* `tabindex="number"` *Встановлює порядок отримання фокусу при переході між елементами за допомогою клавіші **Tab***.
* `title="text"` *Вказує додаткову текстову підказку*.
* `translate="yes|no"` *Повідомляє браузеру переводити вказаний текст в елементі чи ні*.
```
## Події

***Подія*** - *це сигнал від браузера про те, що щось відбулося, наприклад, Ви натиснули клавішу*.

*Починаючи з **HTML4** з’явилась можливість завдяки подіям запускати сценарії в браузері, наприклад: запуск JavaScript, коли користувач натискає на елемент*.

*Нижче наведені загальні атрибути подій, котрі можуть бути додані до HTML-елементів для визначення дії події*.

### Нові події, які з’явились в HTML5
```html
* `onplay="script"` *Скрипт викликається коли медіа дані готові почати відтворення*.
* `onafterprint="script"` *Скрипт виконується тільки після як документ надрукований*.
* `onbeforeprint="script"` *Скрипт виконується перед тим, як документ надрукований*.
* `onbeforeunload="script"` *Скрипт виконується коли документ ось-ось буде вивантажений*.
* `onhashchange="script"` *Скрипт виконується коли там відбулися зміни до частини якоря в **URL***.
* `onload="script"` *Викликається після того як завантаження елемента завершене*.
* `onmessage="script"` *Скрипт виконується коли викликане повідомлення*.
* `onoffline="script"` *Спрацьовує коли браузер починає працювати в автономному режимі*.
* `ononline="script"` *Спрацьовує коли браузер починає працювати в режимі онлай*.
* `onpagehide="script"` *Скрипт виконується коли користувач переходить на іншу сторінку сторінку*.
* `onpageshow="script"` *Скрипт виконується коли користувач заходить на сторінку*.
* `onpopstate="script"` *Скрипт виконується коли змінено історію одного вікна*.
* `onresize="script"` *Скрипт виконується, коли розмір вікна браузера змінюється*.
* `onstorage="script"` *Скрипт виконується, коли вміст Web Storage оновлюється*.
* `onunload="script"` *Викликається, коли сторінка розвантажена, або вікно браузера було зачинено*.
* `onblur="script"` *Скрипт виконується, коли елемент втрачає фокус*.
* `oncontextmenu="script"` *Скрипт виконується коли викликається контекстне меню*.
* `onfocus="script"` *Викликається в той момент, коли елемент отримує фокус*.
* `oninput="script"` *Скрипт викликається коли користувач вводить дані поле*.
* `oninvalid="script"` *Скрипт виконується, коли елемент недійсний*.
* `onreset="script"` *Викликається, коли натискається у формі кнопка типу **Reset***.
* `onsearch="script"` *Викликається, коли користувач щось пише в поле пошуку (для **<input type="search">**)*.
* `onselect="script"` *Викликаєтсья після того як будь-який текст був обраний в елементі*.
* `onsubmit="script"` *Викликається при відправленні форми*.
* `onkeydown="script"` *Подія викликається, коли користувач затис (натиснув та не відпускає) клавішу*.
* `onkeypress="script"` *Викликається коли корисрувач натиснув на клавішу*.
* `onkeyup="script"` *Викликається коли користувач відпускає клавішу*.
* `ondblclick="script"` *Виникає при подвійному клацанні ЛКМ на елементі*.
* `ondrag="script"` *Періодично викликається при операції перетягування*.
```
